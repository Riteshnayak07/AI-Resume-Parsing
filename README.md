# AI Resume Parser & ATS Analyzer

An intelligent Resume Parser that extracts candidate information from resumes and performs ATS scoring, skill extraction, job-role matching, and AI-powered resume analysis.

## Features

- Resume Parsing (PDF, DOCX, TXT)
- Contact Information Extraction
- Skill Detection & Categorization
- ATS Score Calculation
- Job Role Matching
- Resume Comparison
- TF-IDF Keyword Extraction
- NLP-based Text Analysis
- AI-Powered Resume Insights

## Technologies Used

- Python
- Flask
- NLTK
- Scikit-learn
- TF-IDF
- Machine Learning
- NLP
- PyMuPDF
- JSON
